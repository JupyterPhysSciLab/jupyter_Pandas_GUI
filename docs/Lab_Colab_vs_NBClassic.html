Introduction

The examples in the documentation are all based on how the GUI tools work in the classic Jupyter notebook environment. If you are using these tools in Jupyter Lab or Google Colab keep in mind the differences in behavior described below.

Differences common to Lab and Colab

  • Rather then generate the code directly in a notebook cell immediately below the tabbed GUI, the code is generated in an editable textarea below the tabbed GUI. See the image below for the new_pandas_column_GUI().
  • If you wish the generated code to persist after cell outputs are cleared, the generated code needs to be manually copied to a code cell.

Running the code in Lab and Colab

The code can be run by clicking on the "run code" button and the results will appear in the output of the cell. To make sure that the results can be reproduced if the cell outputs are cleard, copy the code that was run from the collapsed "code that was run" summary to a separate code cell.
